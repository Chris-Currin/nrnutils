========
nrnutils
========

This package contains some Python tools for use with the NEURON simulator
(http://www.neuron.yale.edu/).

Original repo: https://bitbucket.org/apdavison/nrnutils/

Author
------------
Andrew Davison

Installation
------------

    $ tar xzf nrnutils
    $ cd nrnutils
    $ python setup.py install
